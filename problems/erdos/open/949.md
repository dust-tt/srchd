## Definitions

Let $S \subseteq \mathbb{R}$ be a set containing no solutions to $a + b = c$.
Must there be a set $A \subseteq \mathbb{R} \setminus S$ of cardinality continuum such that
$A + A \subseteq \mathbb{R}\setminus S$?
