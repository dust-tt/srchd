## Definitions

Let `q : ℕ → ℕ` be a strictly increasing sequence of primes such that
`q (n + 1) - q n ≥ q n - q (n - 1)`. Must `lim q n / (n ^ 2) = ∞`?

Let `q : ℕ → ℕ` be a strictly increasing sequence of primes such that
`q (n + 1) - q n ≥ q n - q (n - 1)`. Then `liminf q n / (n ^ 2) > 0.352`, and this is proved in
[Ri76].
