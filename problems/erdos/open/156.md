## Problem Statement

Does there exist a maximal Sidon set $A\subset \{1,\ldots,N\}$ of size $O(N^{1/3})$?

## Background

A question of Erdős, S&#225;rk&#246;zy, and S&#243;s [[ESS94]](javascript:void(0);). It is easy to prove that the greedy construction of a maximal Sidon set in $\{1,\ldots,N\}$ has size $\gg N^{1/3}$. Ruzsa [[Ru98b]](javascript:void(0);) constructed a maximal Sidon set of size $\ll (N\log N)^{1/3}$.
