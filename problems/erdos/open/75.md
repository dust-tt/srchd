# Erdős Problem 75

## Metadata

- **Status:** open
- **Tags:** graph theory, chromatic number
- **Source:** [erdosproblems.com/75](https://www.erdosproblems.com/75)

## Problem Statement

Is there a graph of chromatic number $\aleph_1$ such that for all $\epsilon&#62;0$ if $n$ is sufficiently large and $H$ is a subgraph on $n$ vertices then $H$ contains an independent set of size $&#62;n^{1-\epsilon}$?

## Background

Conjectured by Erdős, Hajnal, and Szemer&#233;di [[EHS82]](javascript:void(0);).
