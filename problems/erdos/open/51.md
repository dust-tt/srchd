## Problem Statement

Is there an infinite set $A \subset \mathbb{N}$ such that for every $a \in A$,
there is an integer n such that $\phi(n)=a$, and
yet if $n_a$ is the smallest such integer, then $\frac{n_a}{a} → \infty$ as $a → ∞$?
