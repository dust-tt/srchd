# Erdős Problem 70

## Metadata

- **Status:** open
- **Tags:** graph theory, ramsey theory, set theory
- **Source:** [erdosproblems.com/70](https://www.erdosproblems.com/70)

## Problem Statement

Let $\mathfrak{c}$ be the ordinal of the real numbers, $\beta$ be any countable ordinal, and $2\leq n&#60;\omega$. Is it true that $\mathfrak{c}\to (\beta, n)_2^3$?

## Background

Erdős and Rado proved that $\mathfrak{c}\to (\omega+n,4)_2^3$ for any $2\leq n&#60;\omega$.
