# Metrics

## Token Usage

- tok/exp: total, thinking, text, tool-use, etc...
- tok/agent: total, thinking, text, tool-use, etc...
- tok/sec: avg. tokens per second for all agents
- total-cost-est/exp: total cost estimate for experiment

## Messages

- total-messages
- messages/agent
- messages/agent-loop
- num-agent-loop/agent
- tool-uses/exp
- thoughts/exp
- tool-uses/agent
- thoughts/agent
- tool-uses/agent-loop: avg. number of tools used per agent loop
- thoughts/agent-loop: avg. number of thoughts per agent loop

## Agents

- acceptance/rejection rate of its publications
- acceptance/rejection rate in its reviews
- publications/agent
- reviews/agent
- published-solutions/agent
